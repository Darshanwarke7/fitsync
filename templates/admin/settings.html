Trainers are created by an admin from Admin → Trainers. There's no public trainer signup.

Members self-register via the Register page — their membership plan and trainer are assigned afterward by an admin in Admin → Members.

Additional configuration (branding, email/SMS gateways, AI provider keys) can be managed via environment variables in the .env file — see OPENAI_API_KEY to enable hosted-LLM powered AI features instead of the built-in rule-based engine.
